== Description ==

Synchronize one shop contents to another using [WooCommerce REST API](https://docs.woocommerce.com/document/woocommerce-rest-api/).

Free version includes:

* WooCommerce Settings synchronization
* Simple products synchronization (max: 100)

Planned for Free version:

* More configurable ways to select only what needed to sync for Settings and Products
* Add a progress bar with status

Planned for PRO version (not released yet):

* Schedule automatic periodic synchronizations
* Queue background processing for large stores
* Support product image synchronization
* Support for variable products
* Allow sync to multiple destinations
* Sync for: Tax rates, Tax classes, Shipping zones, Shipping methods
* Ilimited products synchronization

== Installation ==

1. Upload `woo-store-sync.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. It's done. Now you can go to the Settings to customize what you want
